/signin (GET)
param   redirect=signInToContinue for add/update/delete MC without signed in
                 noSuchUser       for wrong username
                 passwordWrong    for wrong pasword
return  the sign in page signin.html


/signup (GET)
param   redirect=usernameTooLong  for username.length>12
                 userAlreadyExist for username already exists
return  the sign up page signup.html





/ajax/MC (GET)
param   user&tags (view and page are handled by frontend)
return  _id, title, desc, A, B, C, D, ans, tags, uid, time

/ajax/addMC (POST)
AJAX POST to add a MC, note that _id (question ID), uid (user ID), time are generated bynode.js using session
param   title, desc, A, B, C, D, ans, tags
return  _id, title, desc, A, B, C, D, ans, tags, uid, time

/ajax/updateMC (POST)
AJAX POST to update a MC, note that time are generated by node.js using session, user authwill be checked using session
param   _id, title, desc, A, B, C, D, ans, tags
return  _id, title, desc, A, B, C, D, ans, tags, uid, time
        or 'not authorised'
        or /signin?redirect=signInToContinue

/ajax/deleteMC (POST)
AJAX POST to update a MC, note that user auth will be checked using session
param   _id
return  'succeeded'
        or 'not authorised'
        or /signin?redirect=signInToContinue

/auth/signin (POST)
param   username, password
return  /?redirect=signInSucceeded
        or /signin?redirect=noSuchUser
        or /signin?redirect=passwordWrong

/auth/signout (GET)
return  /?redirect=signOutSucceeded

/auth/signup (POST)
param   username, password
return  /?redirect=signUpSucceeded
        or /signup?redirect=usernameTooLong
        or /signup?redirect=userAlreadyExist
